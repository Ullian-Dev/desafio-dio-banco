# DESCRIÇÃO

Neste projeto, elaborei um banco, contendo suas "principais" ações.

Foi desenvolvido juntamente com o professor Venilton Falvo Jr. 

Para isso, foi explorado o conhecimento em POO, utilizando de suas principais bases (Abstração, Encapsulamento, Herança e Polimorfismo)

Pretendo atualizar esse projeto com algumas ideias que tive para implementar, como por exemplo: Um valor inicial em uma conta poupança, métodos para abrir e fechar uma conta e condições para que isso seja possível (impossibilitando o fechamento de uma conta, caso esteja negativa)
